# Catatan Day 01

## Yang Saya Pahami

- Variable digunakan untuk menyimpan data.
- String digunakan untuk teks.
- Integer digunakan untuk angka bulat.
- Float digunakan untuk angka desimal.
- Boolean digunakan untuk nilai benar atau salah.
- Fungsi input() digunakan untuk mengambil data dari user.
- Input dari user secara default bertipe string.
- Jika ingin melakukan operasi matematika, input perlu dikonversi menggunakan int() atau float().
- f-string memudahkan penulisan output yang berisi variable.

## Kendala

- Masih perlu membiasakan diri membedakan string dan integer.
- Masih perlu latihan menggunakan f-string.
- Masih belajar menggunakan Git dan GitHub.

## Solusi

- Latihan membuat program kecil.
- Mencoba mengubah input string menjadi integer.
- Membaca error message ketika program gagal dijalankan.
- Membiasakan commit harian ke GitHub.

## Kesimpulan

Hari ini saya memahami dasar awal Python, terutama variable, tipe data, input, output, dan f-string. Materi ini menjadi fondasi untuk mempelajari conditional, looping, dan struktur data pada hari berikutnya.
